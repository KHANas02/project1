HTML

HTML ("HyperText Markup Language") veb-sahifalarni yaratish va tuzish uchun ishlatiladigan markaziy tili. HTML yordamida, matn, rasm, linklar, vid–µo, audiotraklar va boshqalarni sahifalarda joylashtirish mumkin. HTML kod HTML taglari bilan yoziladi. Taglar elementlarga bo'linadi va har bir element boshlanishi () va tugashi () bo'lishi mumkin

CSS

CSS ("Cascading Style Sheets") veb-sahifalarni dizayn qilish uchun ishlatiladigan usuldir. CSS bilan, HTML elementlarining o'lchamini, rangini, shriftini, qalinligini, o'rina chiqishini va boshqa stil xususiyatlarini belgilash mumkin

Javascript

JavaScript, brauzerda veb-sahifalarni interaktiv qilish, dinamik funksiyalar qo'shish va veb ilovalarini yaratishda ishlatiladigan universaldan muhim dasturlash tili. JavaScript, veb-sahifalar orqali brauzer bilan foydalanuvchilar o'rtasida o'zaro amalni boshqarish, foydalanuvchining tashqi va ichki amallarini bajarish va sahifalarni dinamik ravishda o'zgartirish imkoniyatini beradi.

BOOTSRAP

Bootstrap, veb-interfeyssizlik uchun eng mashhur va ko'p ishlatiladigan front-end kerakli kutubxonadirlardan biridir. Bootstrap, HTML, CSS va JavaScript yordamida responsive, mobil qurilmalarga mos va ishlab chiqarishni osonlashtiradigan qisqa va o'rganishga yaroqli kerakli elementlarni yig'ish va stylashni taqdim etadi.
